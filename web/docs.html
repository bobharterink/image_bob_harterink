Introductie

Op deze pagina leg ik mijn proces uit tot aan het eindresultaat, voornamelijk aan de hand van afbeeldingen, screenshots en code. Mijn doel is om je een duidelijk inzicht te geven in de stappen die ik heb genomen en de keuzes die ik heb gemaakt. Als je direct naar de code wilt gaan, kun je in het linker menu scrollen naar het hoofdstuk Site code waar alle code te vinden is. Ik heb geprobeerd dit op dezelfde manier weer te geven als op softwaresites waar je de documentatie kunt bekijken. Hopelijk vind je de uitleg nuttig en leerzaam. Veel plezier!
