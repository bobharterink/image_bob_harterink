Notion

Het grootste deel van de informatie die ik hier presenteer, is ook al gedeeltelijk gedeeld op Notion. Echter, om iedereen een beter inzicht te geven, zal ik het hier in platte tekst uitleggen.

            import Dropzone from "dropzone";

            let myDropzone = new Dropzone("#my-element", {
                paramName: "file",
                maxFilesize: 2,
                accept: function(file, done) {
                    if (file.name == "justinbieber.jpg") {
                        done("Naha, you don't.");
                    } else {
                        done();
                    }
                }
            });

            let myDropzone = new Dropzone("#my-element", {
                addedfile: file => {

                }
            });
