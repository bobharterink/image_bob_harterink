Dropzone Configuration Basics

In this section, we'll go over the fundamentals of configuring Dropzones.

Declarative

            <form action="/target" class="dropzone" id="my-great-dropzone"></form>

            <script>
            Dropzone.options.myGreatDropzone = {
                paramName: "file",
                maxFilesize: 2,
                accept: function(file, done) {
                    if (file.name == "justinbieber.jpg") {
                        done("Naha, you don't.");
                    } else {
                        done();
                    }
                }
            };
            </script>

Imperative

            import Dropzone from "dropzone";

            let myDropzone = new Dropzone("#my-element", {
                paramName: "file",
                maxFilesize: 2,
                accept: function(file, done) {
                    if (file.name == "justinbieber.jpg") {
                        done("Naha, you don't.");
                    } else {
                        done();
                    }
                }
            });

Events

To react to events happening, you need to listen to events. Check out the events section for more information.

Overriding Default Event Handlers

            let myDropzone = new Dropzone("#my-element", {
                addedfile: file => {
                    // ONLY DO THIS IF YOU KNOW WHAT YOU'RE DOING!
                }
            });

For a list of all possible options, refer to the options.js file.
